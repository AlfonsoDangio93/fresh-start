# Dashboard Mockup Guide

When creating a dashboard mockup for the Hommi hero section, include these elements:

## Layout
- Sidebar left (dark #101103) with nav icons
- Main content area (white #FFFFFF)
- Top bar with search + user avatar

## Key panels to show
1. **Overview stats** — 4 cards: Alloggi attivi, Ticket aperti, Risolti oggi, Tempo medio
2. **Ticket list** — Table with columns: Alloggio, Problema, Stato (badge), Tecnico, Tempo
3. **Quick action** — "Nuovo ticket" button in primary orange

## Status badges
- Aperto: orange bg
- In corso: blue bg (#344966)
- Risolto: green bg
- Urgente: red bg

## Visual style
- Clean, minimal, lots of whitespace
- Data-dense but readable
- Use Poppins for headings, Open Sans for data
- Subtle shadows, 1px borders
- Charts: simple bar or line, not complex
